Accessibility statement for Find MoJ data website

This accessibility statement applies to the Find MoJ data website at find-moj-data.service.justice.gov.uk. It does not cover other sites or content Find MoJ data links to.

This website was built by the Find MoJ data team at the Ministry of Justice (MoJ) using a template developed by the Government Digital Service (GDS). The Find MoJ data team is responsible for the content of this website.

What this website should do

We want as many people as possible to be able to use this website. For example, that means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 400% without the text spilling off the screen
  • navigate the site using just a keyboard
  • navigate the site using speech recognition software
  • listen to the site using a screen reader (including recent versions of JAWS, NVDA and VoiceOver)

We have also made the text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

Feedback and contact information

If you have problems using this site or need additional support, contact the Find MoJ data team by email at findmojdata@justice.gov.uk.

We'll consider your request and get back to you in 3 working days.

Reporting accessibility problems with this site

We're always looking to improve the accessibility of this site. If you find any problems or think we're not meeting accessibility requirements, contact the Find MoJ data team by email at findmojdata@justice.gov.uk.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the 'accessibility regulations'). If you're not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Technical information about this site's accessibility

We are committed to making this site accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Compliance status

This website has been tested against the Web Content Accessibility Guidelines (WCAG) 2.2 AA standard.

It is partially compliant with the Web Content Accessibility Guidelines version 2.2 AA standard, due to the non-compliances listed below.

Non-accessible content

The following content on Find MoJ data is not compliant with the WCAG version 2.2 AA standard.

Non-compliance with the accessibility regulations

Level A failures

  • Breadcrumb navigation links are announced the same way as other links to screen reader users, who may not mistake breadcrumbs for the main navigation menu (WCAG success criterion 1.3.1 and 4.1.2)

Content outside the scope of the accessibility regulations

PDFs and other documents

We are not responsible for documentation produced and published by other teams. If an entry on Find MoJ data links to PDFs or other inaccessible content, contact the relevant team for alternative versions.

Other services or websites

We are not responsible for other services and websites which Find MoJ data links to. If you have an accessibility concern, check the relevant website's accessibility statement.

Preparation of this accessibility statement

This statement was prepared on 28 March 2025. It was last reviewed on 03 April 2025.

This website was last tested on 25 March 2025 against the WCAG 2.2 AA standard.

The most viewed pages were tested using automated testing tools by our website team in August 2024.

A further accessibility audit was conducted by external specialists during March 2025 against the WCAG 2.2 AA standard.
